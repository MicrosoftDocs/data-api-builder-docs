# DAB Agent extension

The DAB Agent extension provides a Copilot chat participant (`@dab`) for managing DAB workflows with natural-language prompts.

## What it enables

- Initialize and configure DAB projects.
- Add entities and relationships.
- Run validation and runtime tasks.
- Query database schema metadata to guide configuration.

## Tooling model

The extension exposes a DAB CLI executor and schema discovery capability so chat workflows can perform concrete, auditable actions.
